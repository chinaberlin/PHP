# 标题
1-6个#号代表标题

# 区块引用 Blockquotes

\> 代表引用区块，引用可以嵌套，也可在引用里使用其他Markdown语法

# 换行
在结尾出
一行的末尾使用两个以上的空格

# 列表
无须列表 *，有序列表 数字，符号后面要留一个空格。

# 代码区块
只要简单地缩进 4 个空格

# 代码
要标记一小段行内代码，你可以用反引号（`）把它包起来

# 分隔线
一行中用三个以上的星号、减号、底线来建立一个分隔线

# 链接
\[an example](http://example.com/ "Title")

# 强调
星号（*）和底线（_）作为标记强调字词的符号

# 图片
\!\[Alt text](/path/to/img.jpg "title")
